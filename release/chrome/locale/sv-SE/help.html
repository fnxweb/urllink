URL Link

Användning

Markera ett litet stycke text på en webbsida, i en textruta (webbformulär) eller e-brev, och högerklicka.

  • Om texten ser ut som en URL, kan du öppna den direkt i samma fönster eller i en flik via alternativet “Öppna markering”.
  • Om texten inte ser ut som en länk, kommer du att få en undermeny som låter den behandlas som en länk i alla fall, eller som först utför någon anpassningsbar konvertering (t.ex. lägger till “www.” och “.se”).
  • Under Inställningar kan du anpassa undermenyn som dyker upp. Du kan till exempel lägga till egna avskiljare med poster som endast består av bindestreck: "--".
  • Anpassade menyposter kan innehålla vilken URL som helst, oavsett längd, där den första asterisken '*' ersätts av markeringen. Du kan ange ett namn på posten före ett vertikalstreck '|' för att korta ner det som syns i menyn:
    "Kort namn|http://www.ettmycketlångtnamn.se/q=*".
  • I Firefox kan du trycka på <Skift> när du väljer 'Öppna' för att öppna länken i ett nytt fönster istället för att använda det aktuella fönstret.
  • Du kan genom en inställning välja om URL Link skall placera sina alternativ nära toppen eller botten av standardsnabbmenyn.

Funktioner

  • URL Link är ett litet Firefox- och Thunderbird-tillägg som låter dig markera en icke-URL i ett e-brev/RSS-meddelande eller på en webbsida, och öppna den i ett webbläsarfönster.
  • I e-brev fogar det samman länkar som har brutits sönder på flera ställen, och det ersätter också mellanslag med URL-teckenkoden %20 så att du kan följa e-postade `file:'-nätverkslänkar (vilka auto- matiskt identifieras från Windows X:- eller UNC-nätverksreferenser).
  • Också på webbplatser låter tillägget dig markera textuella länkar/URL:er på webbsidor eller i redigeringsrutor, och följa dem som om de var riktiga länkar. Det kommer alltid att låta dig följa länkar, men även analysera mailto:-länkar.
  • URL Link plockar isär `mailto:'-länkar (exempelvis kan den konvertera mailto:vemsomhelst@enwebbplats.se till http://www.enwebbplats.se), så att du kan besöka en närbesläktad webbplats.
  • Tillägget kan `följa' javascript:-länkar som annars skulle ge upphov till ett popup-fönster (t.ex. http://www.kelkoo.co.uk/ - fast de har precis ändrat på några av sina länkar så att URL Link misslyckas :-(. Jag hoppas att kunna ta itu med det efter att version 2.0 har släppts).

Anpassningsbara sök och ersätt-funktioner

Anpassade sök och ersätt-funktioner kan tillämpas på markeringen. Standard- inställningarna är två värden som fanns i tidigare versioner av URL Link (för Thunderbird) och som nu kan utökas eller ersättas. Ersättningen utförs efter att de inledande & avslutande skräptecknen har blivit borttagna, och efter att alla omvända snedstreck har konverterats till vanliga snedstreck.

Formatet för posterna under fliken “Redigera Sök & Ersätt-strängar” är “sökningsRegExp||Ersättning||RegExpFlaggor” (RegExp står för reguljära uttryck). Det sista “||” med efterföljande flaggor är valfritt; den (JavaScript-)flagga som troligast kan vilja använda är 'g' för “global” (eller “ersätt alla”). Ersättningen kan innehålla de vanliga “$n”-sökmönsterreferenserna.

Exempel: Du får regelbundet e-brev med Windows-fillänkar som “\\server\deladefiler\fil”. Men du sitter vid en Linux-dator med Windows delade filer mappade till “/mnt/smb/användarnamn/deladefiler”. Då lägger du till en post i slutet av listan (märk att omvända snedstreck redan har konverterats och att standardposterna konverterar “//server/deladefiler” till “file:///server/deladefiler”) i formatet: “file:///server/deladefiler||file:///mnt/smb/användarnamn/deladefiler”.

Om du behöver felsöka dina uttryck, lägg till URL Links särskilda RegExp- flagga “D”, som i: ‘från||till||D’. Då kommer URL Link att öppna en ruta som talar om vad ditt uttryck är (“Konvertering:”), vad markeringen är för tillfället (“Från:”) och vad uttrycket har gjort med markeringen (“Till:”).

Framtida förändringar

  • Utöka de anpassningsbara sök och ersätt-funktionerna när tillägget är stadigt nog att också tillåta slumpmässiga JavaScript-koder att köras över markeringen.
  • Kanske flytta några av objekten i den anpassade undermenyn till huvudmenyn.
