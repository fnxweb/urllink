URL Link

Utilizzo

Permette di aprire nella stessa scheda oppure in una nuova selezioni del testo e link testuali presenti nelle pagine web e link testuali presenti in messaggi di posta elettronica o di gruppi di discussione tramite clic con tasto destro del mouse e selezione di una fra le voci aggiunte da URL Link

  • Se il testo è un link selezionabile, si potrà aprirlo direttamente (o in una nuova scheda) tramite “Apri selezione”
  • Se il testo non è un link selezionabile, si aprirà un sottomenu che permetterà di aprire il testo selezionato come un vero link, o di effettuare prima alcune conversioni personalizzabili (es: Con “www.*.com”)
  • Nella finestra delle opzioni, dove è possibile determinare la posizione degli elementi di menu, è possibile anche aggiungere separatori i cui elementi consistono in semplici trattini. Es: "--"
  • Le voci di menu personalizzate possono contenere qualsiasi URL, indipendentemente dalla lunghezza. Il primo segno '*' verrà sostituito dalla selezione. È possibile abbreviare ciò che apparirà nel menu inserendo un alias nell'elemento prima della barra '|'
    Es: "Nome breve|http://www.longname.com/q=*"
  • In Firefox, tenere premuto il tasto <Maiusc> quando si seleziona 'Apri' per aprire il link in una nuova finestra invece di aprirlo in quella attuale
  • Nella finestra delle opzioni è possibile selezionare 'Mostra in cima al menu contestuale' per posizionare le voci di menu di URL Link in cima al menu contestuale

Funzionalità

  • URL Link è una piccola estensione per Firefox e Thunderbird che permette di selezionare un "non URL" in un messaggio di posta elettronica o in una pagina web e di aprirlo in una finestra del browser
  • Per i messaggi di posta elettronica, ripristina quei link interrotti da parecchie linee e sostituisce gli spazi degli indirizzi web con il codice "%20" affinché sia possibile aprire i link di file in rete (che sono rilevati automaticamente da Windows come "X:" o come riferimenti di rete UNC)
  • Per le pagine web, permette di selezionare link testuali o URL nelle pagine web o nei campi di testo ed aprirli sempre come fossero veri link. URL Link permette inoltre di analizzare i link di tipo "mailto:"
  • URL Link permette di analizzare i link di tipo "mailto:" convertendoli (es: mailto:someone@somesite.com diventa http://www.somesite.com) affinché sia possibile visitare il sito web relativo
  • URL Link permette di aprire anche link di tipo JavaScript che altrimenti aprirebbero pop-up (es: http://www.kelkoo.co.uk/ - anche se recentemente sono stati modificati alcuni dei link ed essi non sono più apribili direttamente)

Stringhe di ricerca/sostituzione

È possibile applicare le stringhe di ricerca/sostituzione alla selezione. Le impostazioni predefinite corrispondono a due valori presenti nelle precedenti versioni di URL Link (per Thunderbird) e che attualmente possono essere aggiunti o sostituiti: la sostituzione avviene dopo l'eliminazione di porzioni di codice in eccedenza all'inizio ed alla fine dell'espressione regolare e dopo la conversione di tutti i segni grafici “\\” nel segno grafico “//”

Il formato dell'elemento nella finestra delle impostazioni corrisponde a “searchRegExp||Replacement||regExpFlags”. I segni grafici finali “||” e gli indicatori sono opzionali; l'indicatore di JavaScript probabilmente più utile corrisponde a 'g' per “global” (oppure “replace-all”). Nell'espressione per la sostituzione potrebbero essere presenti i riferimenti usuali “$n” del modello di ricerca

Es: si ricevono e-mail contenenti link di Windows a file come “\\server\share\file”. Se si è in ambiente Linux si avrebbe una finestra di condivisione mappata come “/mnt/smb/username/share”. È possibile aggiungere un elemento alla fine della lista (da notare che i segni grafici “\\” sono già stati convertiti e che gli elementi predefiniti convertono “//server/share” in “file:///server/share”) del modulo “file:///server/share||file:///mnt/smb/username/share”

Per effettuare il debug delle espressioni basta aggiungere (in URL Link) all'espressione regolare lo speciale indicatore “D” nella seguente maniera: ‘From||To||D’. In questo modo URL Link aprirà una finestra di avviso contenente i dati attuali dell'espressione regolare (“Conversion:”), a che punto si trova la selezione (“From:”) e quali effetti ha prodotto l'espressione regolare alla selezione (“To:”)

Prossime modifiche

  • Estensione di modifica di ricerca e sostituzione di stringhe quando è possibile permettere in maniera stabile codice JavaScript arbitrario per la selezione
  • Possibilità di spostare alcuni dei sottomenu personalizzati direttamente nel menu contestuale
