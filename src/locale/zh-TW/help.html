URL Link

用法

在網頁，文字框或電郵中選取文字, 然後按右鍵。

  • 如果文字像是一個網址，您可以經"開啟選項"直接開啟它 (或開啟到分頁)。
  • 如果文字不像是一個網址，會有子選單將該文字變成一個網址； 或執行一些自訂的轉換方式 (如加上“www.”和“.com”)。
  • 在選項，您可以自訂子選單的項目，加只有連字號(hyphen)的分隔符號，如 "--"。
  • 自訂子選單項目可能包含URL，無論多長，第一個"*"將會先被取代。您可以在"|"前加別名，顯示就會縮短。
    "Short name|http://www.longname.com/q=*" 就會只顯示 "Short name"。
  • 在 Firefox 中, 選擇開啟時按住 <Shift> 可以開啟到一個新視窗。
  • 您可以經"選項"，選擇 URL Link 放在右鍵選單的位置。

功能

  • URL Link 是一個小巧的 Firefox 和 ThunderBird 擴充套件， 可以將一段在電郵、新聞、網頁裡不像網址的文字，開啟到視窗中。
  • 在電郵，它再連接已斷開幾行的電郵鏈結； 將字元碼 %20 代替空白鍵，令您可以追蹤電郵網絡 "file:" 鏈結 (從 WindowsX 開始自動偵測；或通用命名標準[UNC]的網絡參照)。
  • 在網頁，它可以讓純文字的鏈結或網址，成為真正鏈結。 它可以時常追蹤鏈結並同時分析 mailto: 鏈結。
  • 可以分割 mailto: 鏈結 (如將 mailto:someone@somesite.com 分開為 http://www.somesite.com)， 令您可以瀏覽相關網址
  • 可以追蹤 javascript: 可以產生彈出視窗的鏈結。 (如 http://www.kelkoo.co.uk/ - 儘管他們改變了一些鏈結令其失效 :-( 我希望到2.0版時可以有效。)

將來

  • 可自訂自動用網址代替文字
  • 使一些自訂的副選單項目上移到主選單。
