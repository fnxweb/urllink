URL Link

用法

在網頁，文字框或電郵中選取文字, 然後按右鍵。

  • 如果文字像是一個網址，您可以經開啟選項直接開啟它 (或開啟到分頁)。
  • 如果文字不像是一個網址，會有個子選單去令到該文字是一個網址； 或執行一些自訂的轉換方式 (如加“www.”和“.com”)。
  • 在 Firefox 中, 選擇開啟時按住 <Shift> 可以開啟到一個新視窗。

功能

  • URL Link 是一個小巧的 Firefox 和 ThunderBird 擴充套件， 可以開啟一段在電郵，網頁裡不像網址的文字到視窗中。
  • 在電郵，它再連接已斷開的電郵鏈結； 將字元碼 %20 代替空白鍵，令您可以跟蹤電郵網絡檔案鏈結 (從WindowsX開始自動偵測；或通用命名標準[UNC]的網絡參考)。
  • 在網頁，它可以讓純文字的鏈結或網址，成為真的鏈結。 它可以時常跟隨鏈結並同時分析 mailto: 鏈結。
  • 可以分割 mailto: 鏈結 (如將mailto:someone@somesite.com 分開為http://www.somesite.com)， 令您可以瀏覽相關網址
  • 可以追蹤javascript: 可以產生彈出視窗的鏈結。 (如http://www.kelkoo.co.uk/ - 儘管他們改變了一些鏈結令其失效 :-( 我希望到2.0版時可以有效。)

將來

  • 可自訂自動用網址代替文字
  • 使一些自訂的副選單項目上移到主選單。
